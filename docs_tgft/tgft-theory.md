# TGFT：术语与范围提示（避免歧义）

本目录文档里出现的 “TGFT” 缩写，容易与学术界常用的 **Tensorial Group Field Theory（同缩写 TGFT）** 混淆。这里用一页说明把二者区分开，避免读者把“自定义叙述”误当作主流 TGFT 文献结论。

## 1. 主流语境下的 TGFT（Tensorial Group Field Theory）

在主流研究中，TGFT 通常指张量化群场论/张量群场论一类量子引力路径：强调张量模型、群场论、重整化群流与相变中“时空涌现”等主题。

## 2. 本目录文档中的 “TGFT”（本仓库用法）

本目录其它文件（如 `tgft-general-relativity.md`、`tgft-standard-model.md` 等）把 “TGFT” 用作一个**统一规范场论式**的组织方式（常见元素包括标架场 `e^a_μ`、自旋联络 `ω^{ab}_μ`、以及 `PG(1,3)=SP(1,3)⋊W^{1,3}` 这类写法）。这套叙述与主流 Tensorial GFT 的标准定义**并不等价**，更接近一种探索性“统一框架讲义”。

## 3. 建议阅读方式

- 如果你的目标是**理解本仓库这套叙述内部的一致性**：从 `tgft-unified-framework.md` 的索引进入即可。
- 如果你的目标是**对齐主流 TGFT（Tensorial GFT）研究**：请以该方向的专门教材/论文为准，并将本目录内容视为独立的探索性文本。
