# TGFT-EXT 推演（工作稿）

> 目标：在不牺牲一致性（稳定性/因果性/实验约束）的前提下，把 TGFT-EXT 从“叙述”推进到**可变分、可计算、可检验**的模型。

## 0. 阅读说明

- 本文是 **TGFT-EXT 的增量推演**入口；基础材料请先看 `docs_tgft/index.md`。
- 公式渲染：本仓库采用 GitHub 支持的 `$...$` 与 `$$...$$`。

## 1. 最小可检验版本（MVP）定义

### 1.1 基本自由度

- **标架场**：$e^a{}_\mu$（$e=\det(e^a{}_\mu)$）。
- **联络**：$\omega^{ab}{}_\mu$（是否独立变分：是）。
- **物质场**：用 $\psi$ 统称。
- （可选扩展）**额外场**：$\Phi$（标量/伪标量/矢量，需明确其对称性与耦合）。

### 1.2 对称性（必须逐条写清）

- 局域洛伦兹对称性（tetrad formalism）。
- 微分同胚不变性（坐标协变）。
- 额外：若引入 $\Phi$，其内部对称性与宇称/时间反演性质。

### 1.3 作用量（占位：后续只允许“可检验扩展项”）

我们以一阶形式为起点（Einstein–Cartan / Palatini 路线），先写一个统一模板：

$$
S = \int d^4x\, e\,\Big[\underbrace{\frac{1}{2\kappa}R(e,\omega)}_{\text{引力基底}}\; +\; \underbrace{\mathcal{L}_m(e,\omega,\psi)}_{\text{物质}}\; +\; \underbrace{\mathcal{L}_{\text{EXT}}(e,\omega,\Phi)}_{\text{扩展}}\Big].
$$

其中：
- $\kappa = 8\pi G$。
- $R(e,\omega)$：由 $\omega$ 的曲率构造（需要明确指标约定与符号）。

**扩展项选择原则（硬约束）**：
- 不引入 ghost（高阶导导致的 Ostrogradsky 不稳定），或给出有效场论截止与稳定性证明。
- 弱场极限必须回收 GR 的 PPN 约束（至少在参数空间一部分满足）。
- 与等效原理/宇宙学/引力波约束相容（明确哪些耦合会破坏普适自由落体）。

## 2. 一致性检查清单（写在模型之前，而不是写在模型之后）

### 2.1 线性化谱（Minkowski / FRW 背景）

- 传播自由度：自旋-2、自旋-1、自旋-0 的质量与符号。
- 无 ghost：动能项符号正确。
- 无梯度不稳定：声速平方 $c_s^2>0$。
- 因果性：无超光速或给出可接受解释（例如 EFT 范围内）。

### 2.2 能量条件与“反重力”语义澄清

“反重力”在本文中必须被分解为三类之一：
- (A) **净排斥**：对普通物质出现等效势为正的长程排斥。
- (B) **屏蔽/衰减**：有效 $G$ 在某区域被降低。
- (C) **第五力**：对某种新“引力电荷”呈相反符号耦合。

只有在选定 (A)/(B)/(C) 后，才讨论实现机制与可观测。

### 2.3 实验约束（最小集合）

- 自由落体等效原理（Eötvös 参数）。
- 太阳系 PPN 参数。
- 引力波传播速度与色散。
- 第五力/屏蔽机制实验（短程与天文尺度）。

## 3. 推演方向 I：人造重力场

### 3.1 定义目标

- **真实引力**：修改度规/联络导致的真实测地线偏转。
- **等效引力**：加速/旋转/模拟体系产生的等效重力（不等同于改写引力场）。

### 3.2 计算入口（待填）

- 从场方程线性化得到静态势 $\phi(r)$，并计算测试粒子的加速度 $\mathbf{a}=-\nabla\phi$。
- 对给定源的 $T_{\mu\nu}$，估算达到 $|\mathbf{a}|\sim g_\oplus$ 所需能量密度与尺度。

## 4. 推演方向 II：反重力/排斥机制

### 4.1 结构性判据（占位）

- 在“普适耦合 + 健康自由度”的前提下，长程净排斥往往不可得；若要得到排斥，通常需要：
  - 非普适耦合（引入新电荷/屏蔽机制），或
  - 有效能量条件被破坏（并给出稳定实现），或
  - 特定背景场导致的局部有效势改变。

### 4.2 可检验信号

- 等效原理破坏（组分依赖加速度）。
- 第五力 Yukawa 项：$V(r)\sim \alpha\,e^{-mr}/r$ 的实验上限对参数的约束。

## 5. 推演方向 III：黑洞理论重构

### 5.1 解的层级（按可观测排序）

1. 静态球对称：是否偏离 Schwarzschild。
2. 旋转解：是否偏离 Kerr。
3. 线性扰动：QNM（振铃频率）与衰减率。
4. 热力学：熵、温度、第一定律是否修正。
5. 信息问题：若有机制，必须落到可计算量（例如纠缠熵修正、辐射谱修正）。

### 5.2 计算入口（待填）

- 给定 TGFT-EXT 场方程，设球对称 ansatz：

$$
 ds^2 = -f(r)dt^2 + f(r)^{-1}dr^2 + r^2 d\Omega^2.
$$

- 求 $f(r)$ 的修正项并提取：
  - 地平线位置 $r_h$，
  - 光子球/ISCO，
  - QNM 频谱的 leading 修正。

## 6. 下一步要做的“硬产出”（按优先级）

1. **写死一套 $\mathcal{L}_{\text{EXT}}$（最小项集）**：只保留能被检验的 1–2 个参数。
2. 完成 Minkowski 线性化与传播谱判据。
3. 给出静态势 $V(r)$ 的具体形式与实验上限对参数的约束。
4. 给出球对称黑洞解的 leading 修正与可观测量。
