## 2. 核心数学结构

### 2.1 标架场 e^a_μ

标架场是 TGFT 的核心，连接局部洛伦兹标架与坐标标架：

\[ g_{μν} = e^a_μ e^b_ν η_{ab} \]

- **物理意义**：描述时空几何，对应引力
- **规范群**：W^{1,3} 手征平移群
- **场强**：挠率张量 T^a_{μν}

### 2.2 自旋联络 ω^{ab}_μ

自旋联络是自旋规范群的规范势：

\[ ω^{ab}_μ = -ω^{ba}_μ \]

- **物理意义**：描述非引力相互作用
- **规范群**：SP(1,3) 自旋规范群
- **场强**：曲率张量 F^{ab}_{μν}

### 2.3 统一作用量

TGFT 的统一作用量：

\[ S = S_G + S_{SP(1,3)} + S_m \]

其中：
- S_G：引力作用量（标架场动力学）
- S_{SP(1,3)}：自旋规范作用量（包含电磁、弱、强）
- S_m：物质场作用量
