# Herbiverse 香草植物数据库

## 项目简介

Herbiverse是一个综合性的香草植物数据库项目，致力于收集、整理和分享关于香草植物的详细信息。项目包含植物的形态特征、栽培方法、化学成分、烹饪用途、毒性信息等全方位数据，为园艺爱好者、厨师、研究人员提供可靠的参考资料。

### 项目特色
- **科学准确**: 基于权威植物学数据库和科学文献
- **实用导向**: 包含栽培难度、烹饪应用、储存方法等实用信息
- **安全第一**: 详细标注有毒植物，提供安全使用指导
- **多语言支持**: 提供中英文对照的植物名称和描述
- **结构化数据**: JSON格式便于程序化处理和查询

## 已完成条目列表

### 可食用香草植物 (10种)

1. **罗勒 (Basil)** - *Ocimum basilicum*
   - 难度: 容易（温暖季节）
   - 特色: 意式料理经典，制作青酱的必备香草

2. **薄荷 (Mint)** - *Mentha spp.*
   - 难度: 容易（需控制蔓延）
   - 特色: 清凉感强烈，茶饮和甜点的完美搭配

3. **香菜/芫荽 (Cilantro/Coriander)** - *Coriandrum sativum*
   - 难度: 中等（偏凉爽季节更易成功）
   - 特色: 亚洲料理必备，叶籽两用

4. **欧芹 (Parsley)** - *Petroselinum crispum*
   - 难度: 容易至中等（幼苗期生长缓慢）
   - 特色: 地中海料理基础香草，营养价值高

5. **韭葱/细香葱 (Chives)** - *Allium schoenoprasum*
   - 难度: 极易（新手友好）
   - 特色: 温和葱香，法式料理经典

6. **莳萝/刁草 (Dill)** - *Anethum graveolens*
   - 难度: 容易（耐寒性强）
   - 特色: 北欧料理灵魂，腌制蔬菜必备

7. **牛至/披萨草 (Oregano)** - *Origanum vulgare*
   - 难度: 容易（耐旱耐寒）
   - 特色: 披萨和意面的经典调料

8. **百里香 (Thyme)** - *Thymus vulgaris*
   - 难度: 容易（耐旱耐寒）
   - 特色: 法式香草束核心成员

9. **迷迭香 (Rosemary)** - *Rosmarinus officinalis*
   - 难度: 容易（适应性强）
   - 特色: 松香浓郁，烤肉专用香草

10. **鼠尾草 (Sage)** - *Salvia officinalis*
    - 难度: 容易（耐旱耐寒）
    - 特色: 感恩节火鸡调料，意式黄油酱

### 有毒观赏植物 (2种)

⚠️ **警告: 以下植物有毒，仅供观赏，严禁食用**

1. **夹竹桃 (Oleander)** - *Nerium oleander*
   - 毒性: 全株剧毒，含强心苷
   - 症状: 心律不齐、恶心呕吐、甚至死亡

2. **毛地黄 (Foxglove)** - *Digitalis purpurea*
   - 毒性: 全株含洋地黄毒苷
   - 症状: 视觉异常、心律失常、意识模糊

## 越南粉与东南亚常用香草

### 越南河粉 (Phở) 的香草传统

越南河粉是越南最具代表性的料理之一，其灵魂不仅在于清澈鲜美的汤头和滑嫩的米粉，更在于那一盘新鲜香草的完美搭配。传统的越南河粉香草拼盘通常包含以下几种:

**核心香草组合:**
- **薄荷 (Mint)** - 提供清凉感，平衡汤头的油腻
- **香菜 (Cilantro)** - 增添清香层次，是东南亚料理的经典
- **罗勒 (Thai Basil)** - 使用泰式罗勒，茴香味更浓，耐热性好
- **韭葱 (Chives)** - 提供温和的葱香

**其他常见配菜:**
- **豆芽菜** - 增加爽脆口感
- **柠檬片** - 提供酸味调节
- **小辣椒** - 调节辣度

### 东南亚料理香草文化

东南亚地区的热带气候孕育了丰富的香草文化，新鲜香草在当地料理中扮演着不可或缺的角色:

**泰式料理:**
- 泰式罗勒 (Thai Basil) - 耐热性强，适合热炒
- 柠檬香茅 - 汤品和咖喱的必备
- 泰式薄荷 - 沙拉和冷菜专用

**越南料理:**
- 新鲜香菜 - 几乎所有菜品的标配
- 薄荷叶 - 春卷和河粉的经典搭配
- 紫苏叶 - 烤肉和火锅的完美配菜

**印尼马来料理:**
- 椰浆香草组合 - 配合各种咖喱
- 香兰叶 - 甜点和饮品的天然香料

### 使用建议

1. **新鲜优先**: 东南亚料理强调香草的新鲜度，干燥香草无法替代
2. **现加现用**: 香草应在食用前最后加入，保持最佳香气
3. **平衡搭配**: 不同香草的组合能创造层次丰富的口感
4. **因地制宜**: 根据当地可获得的香草品种进行调整

## 使用说明

### 数据格式

每个植物条目包含以下标准字段:
- `id`: 唯一标识符
- `cn_name`: 中文名称
- `en_name`: 英文名称  
- `scientific_name`: 学名
- `category`: 分类 (edible/non_edible)
- `edible_parts`: 可食部位
- `toxicity`: 毒性信息
- `cultivation`: 栽培信息
- `chemistry`: 化学成分
- `culinary_uses`: 烹饪用途
- `storage`: 储存方法
- `references`: 参考文献

### 索引文件使用

`index.json`提供多种索引方式:

1. **alphabetical_index**: 按英文名A-Z排序
2. **chinese_index**: 按中文拼音排序
3. **category_index**: 按可食性分类
4. **usage_index**: 按用途分类 (烹饪/药用/芳香/观赏)
5. **difficulty_index**: 按栽培难度分类
6. **statistics**: 数据库统计信息

### 安全注意事项

⚠️ **重要提醒**:
- 有毒植物已明确标注，切勿误食
- 使用前请仔细阅读毒性信息
- 如有过敏史，请谨慎使用新香草
- 孕期哺乳期人群请咨询专业医师
